com.shephertz.app42.paas.sdk.csharp.storage.StorageService Class Reference

Storage service on cloud provides the way to store the JSON document in NoSQL database running on cloud. One can store the JSON document, update it , serach it and can apply the map-reduce search on stored documents. Example : If one will store JSON doc {"date":"5Feb"} it will be stored with unique Object Id and stored JSON object will look like { "date" : "5Feb" , "_id" : { "$old" : "4f423dcce1603b3f0bd560cf"}}. This old can further be used to access/search the document. More...
